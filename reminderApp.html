n8n Automation Flow

Setup Required

The n8n automation handles:

  • ● Receiving messages from WhatsApp & Telegram bots
  • ● Parsing meeting details and saving to Supabase
  • ● Scheduling reminders (2-3 notifications, 1hr before meeting)
  • ● Sending reminders via Email, WhatsApp, and Telegram simultaneously

Deploy n8n on Railway (free tier) → Connect Telegram Bot + WhatsApp Business API → Import the provided workflow JSON.
